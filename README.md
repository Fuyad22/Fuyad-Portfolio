# Fuyad Portfolio

A modern, dynamic portfolio website built with Next.js 14, TypeScript, and Tailwind CSS. Features a responsive design with hero section, skills showcase, project gallery, and contact information. Includes an admin panel for easy content updates.

## Features

- **Hero Section**: Eye-catching introduction with profile image and title
- **About Section**: Personal description
- **Skills Section**: Display of technical skills as tags
- **Projects Gallery**: Grid layout showcasing projects with images and descriptions
- **Contact Section**: Links to email, phone, LinkedIn, and GitHub
- **Admin Panel**: Dynamic content management at `/admin`
- **Responsive Design**: Optimized for all devices
- **Dark Mode Ready**: Styled with Tailwind CSS for easy theming

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Admin Panel

Visit [http://localhost:3000/admin](http://localhost:3000/admin) to update portfolio content dynamically.

Access the admin panel at [http://localhost:3000/admin](http://localhost:3000/admin) to update content.
